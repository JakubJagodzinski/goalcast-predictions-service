# GoalCast predictions-service

## Requirements

- Java 17
- Python 3.10
- PySpark 4.1.1
- Add JAVA_HOME to environment variables
- Add %JAVA_HOME%\bin to PATH
- Copy hadoop folder from repository to C:\
- Add HADOOP_HOME=C:\hadoop to environment variables
- Add %HADOOP_HOME%\bin to PATH
- Make sure to close and reopen command prompt window
